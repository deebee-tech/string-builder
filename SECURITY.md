# Security Policy

## Supported Versions

The latest release on npm/JSR (`@deebeetech/string-builder`) is supported for security fixes.

## Reporting a Vulnerability

Please use [GitHub Private Vulnerability Reporting](https://github.com/deebee-tech/string-builder/security/advisories/new) for this repository.

If that is unavailable, email the package author listed in `package.json`. Do not open a public issue for undisclosed vulnerabilities.
